## 345. Reverse Vowels of a String 

Write a function that takes a string as input and reverse only the vowels of a string.

**Example 1:**
<pre>
<b>Input:</b> "hello"
<b>Output:</b> "holle"
</pre>

**Example 2:**
<pre>
<b>Input:</b> "leetcode"
<b>Output:</b> "leotcede"
</pre>

**Note:**<br/>
The vowels does not include the letter "y".
