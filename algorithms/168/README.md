## 168. Excel Sheet Column Title

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:
<pre>
    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
</pre>

**Example 1:**
<pre>
<b>Input:</b> 1
<b>Output:</b> "A"
</pre>

**Example 2:**
<pre>
<b>Input:</b> 28
<b>Output:</b> "AB"
</pre>

**Example 3:**
<pre>
<b>Input:</b> 701
<b>Output:</b> "ZY"
</pre>
