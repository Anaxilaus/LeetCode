## 171. Excel Sheet Column Number

Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:
<pre>
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
</pre>

**Example 1:**
<pre>
<b>Input:</b> "A"
<b>Output:</b> 1
</pre>

**Example 2:**
<pre>
<b>Input:</b> "AB"
<b>Output:</b> 28
</pre>

**Example 3:**
<pre>
<b>Input:</b> "ZY"
<b>Output:</b> 701
</pre>
